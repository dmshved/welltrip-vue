<h2 align="center">

</h2>

The API repo: [welltrip](https://github.com/Dmytro-Shved/welltrip)

### Installation

- Clone the repository

```
https://github.com/dmshved/welltrip-vue.git
```

- Copy `.env.example` file to `.env` and edit database credentials there

```
copy .env.example .env
```

- Install dependencies

```
npm install
```

- Run the application

```
npm run dev
```
